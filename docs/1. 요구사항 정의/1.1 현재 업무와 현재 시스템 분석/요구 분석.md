# 원하는 기능
크롬 오른쪽 상단에서 확장 프로그램 아이콘을 클릭하여 접근.<br/>
아이콘을 클릭하면 작은 팝업이 뜸.<br/>
'Prevent Crying 'T.T'' 문장 아래 ON/OFF 토글 버튼이 있음.<br/>
ON이면 T.T와 같이 우는 표정을 입력하는 것을 막음.<br/>
OFF면 입력에 제한 없어짐.
